📋 使用说明

  1. 依次选择您的区域、建筑、单元和房间号
  2. 系统会自动显示对应的照明和空调房间编号
  3. 点击"复制"按钮可将编号复制到剪贴板
  4. 将复制的编号配置到 GitHub Secrets 中的 LIGHT_ROOM 和 AC_ROOM
